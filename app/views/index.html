PPPT Component Library

A single source of truth for user interface components and interaction patterns across policing services

This site is aimed at interaction designers to help with speedy setup of on brand PPPT prototypes. To get started as quick as possible take a copy of this repo and use it to start creating your prototypes. It is built upon the Home Office internal Design System which is in term built upon GDS's Design System.

Note to developers

This prototype kit is an extension of the base Home Office prototyping kit. All code is not production code. The information held here is just for guidance to designers, business analysts and developers to understand the predefined components available to teams in PPPT.

There is no real data or any integrations with real data. This site is not connected to any APIs or sensitive information policing information. Everything here is for demonstration only.
